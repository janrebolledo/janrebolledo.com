Sustainability

Sustainability has been prioritized in the environmental initiatives of this website, with a focus on maximizing efficiency and minimizing its environmental impact.

Hosting

The website is hosted through Netlify, which leverages the resources of both AWS and Google Cloud.

Additional information on their sustainability efforts can be found by following the respective links.

Size

The website has been built using Astro, a lightweight framework that sends no JavaScript to the client. This approach reduces the load on both the user and servers, which contributes to sustainability efforts.

Content Management System

The Content Management System utilized by the website is Contentful, and while information on its footprint is not publicly shared, it has been selected for its environmental considerations.
